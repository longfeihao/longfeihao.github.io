远程服务器使用SSH连接方法

SSH配置

SSH链接服务器

SSH链接服务器等，本可以直接用ssh name@ip -p 端口号,然后根据提示输入密码，但是有两个缺点

  1. 繁琐，每次联结搜需要输入好长的字符
  2. 安全性不如SSH

生成密钥

全部按Enter，最后会在~/.ssh/下生成id_rsa和id_rsa.pub分别为密钥和公钥

远程服务器连写公钥

然后把公钥复制到authorized_keys文件内

使用SSH联结远程服务器

可以将该服务器存在配置文件内，这样就不要输入上述那么长的命令了

SSH链接GitLab

  • 配置本地git

  • 直接使用上面生成的密钥和公钥，把公钥添加到gitlab网站

  • 验证是否成功

    成功会提示

    Welcome to GIT, 遥光!

  • 已有代码创建仓库

  • 配置
    编辑`~/.ssh/config
    添加GitLab

  • 在每个git仓库中如下配置，可以使用密钥操作

# ssh # git
