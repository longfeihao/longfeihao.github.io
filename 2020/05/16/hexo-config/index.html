文章插入图片

  1. 在主配置文件_config.yaml中将post_asset_folder: false改为true。
  2. 安装插件npm install https://github.com/CodeFalling/hexo-asset-image -- save
  3. 创建博客时会同时生成一个同名文件夹，将图片放入该文件夹中。如hexo new post title会创建title.md和/title
  4. 在title.md插入图片!(note)[title/iamge_file]
